Sokkelossa mönkijä = 'M'.
           robootti = 'R'.
           esine = 'E'.
           seinä = '*'.
           vapaa käytäväpaikka = ' '.

Komennot pelille

kartta = tulosta kartta.
katso p/i/l/e = tulostetaan mönkijän viereisen paikan tiedot.
liiku p/i/l/e = liiku johonkin suuntaan.
inventoi = tulosta mönkijän tiedot.
muunna 'n' = muunnetaan invnetoorista n määrä esineitä energiaksi mönkijälle.
odota = mönkijä pysyy paikoillaan sokkelossa mutta robootit liikkuvat.
lataa = ladataan peli alusta
tallenna = tallennetaan peli (varo! tämä korvaa alkuperäisen sokkelon).
lopeta = lopetetaan peli.
